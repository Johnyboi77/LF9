# Login-Übersicht – LF9 Heiner IT-Systems GmbH

> Alle Passwörter entsprechen dem Abteilungsnamen (Klartext, lt. Aufgabenstellung).

## Test-Logins

| Abteilung | Passwort | Zugriff auf |
|---|---|---|
| Lager | `Lager` | Feature 1 (Verbindung), Feature 2 (SQL), Feature 4 (DB-Tabelle) |
| Verwaltung | `Verwaltung` | Feature 1, Feature 2 (SQL), Feature 3 (CSV), Feature 4 (DB-Tabelle) |
| Marketing | `Marketing` | Feature 1, Feature 2 (SQL), Feature 4 (DB-Tabelle), Feature 5 (XML) |
| Geschäftsführung | `Geschäftsführung` | Alle Features + 3 Umsatz-Diagramme (Dashboard) |

## Hinweis

Das Login-System ist **kein echtes Authentifizierungssystem** – es ist ein Lern-Beispiel für RBAC (Role-Based Access Control). Passwörter sind Klartext und entsprechen dem Abteilungsnamen, wie in der Aufgabenstellung vorgegeben.
